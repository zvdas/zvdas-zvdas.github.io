• Create a new app by running the following command in terminal
    ng new fba_v1
  • Once created, go into the app folder by running the following command in terminal
    cd fba_v1
  • Install bootstrap & json-server by running the following command in terminal
    npm i --save bootstrap json-server
  • Generate the module blog by running the following command in terminal
    ng g m blog -m=app
  • Generate the classes blog & contact inside the blog module by running the following command in terminal
    ng g cl blog/classes/blog/blog --type=model
    ng g cl blog/classes/contact/contact --type=model
  • Generate the services blog & contact inside the blog module by running the following command in terminal
    ng g s blog/services/blog/blog
    ng g s blog/services/contact/contact
  • Generate the components home, blog, contact, show, error & navbar inside the blog module by running the following command in terminal
    ng g c blog/components/home
    ng g c blog/components/blog
    ng g c blog/components/contact
    ng g c blog/components/show
    ng g c blog/components/error
